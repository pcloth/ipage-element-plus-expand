element-plus组件库支持 ​

TIP

element-plus的绝大部分组件都已经配置了支持，你只需要把el-input这样的组件名称，去掉el-，变成input填写到slot字段中即可

例如

html
const item = {
    id: "name",
    slot: "input",
    label: "姓名"
}
