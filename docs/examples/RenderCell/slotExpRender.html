render 渲染函数 ​

当渲染单元的slot="render"的时候，你可以直接传入一个render方法，并使用jsx来直接渲染一个自定义组件。

允许接收一个渲染函数，用来直接渲染内容

这是一个自定义渲染
vue
<template>
    <RenderCell :item="item"/>
</template>
<script lang="jsx">
    export default {
        data(){
            return {
                item:{
                    slot:'render',
                    isFormItem:false,
                    render:(loadData)=>{
                        console.log(loadData,'loadData')
                        return <span>这是一个自定义渲染</span>
                    }
                }
            }
        }
    }
</script>```
